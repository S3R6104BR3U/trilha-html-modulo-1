Como Surgiu?

Anteriomente a redes de computadores vale destacar a comunicação por telegrafo onde países já podiam se comunicar a longas distancias utilizando o código morse. Na Década de 1950 periodo da Guerra Fria foram desenvolvidas as redes de computadores Destaques das primeiras Redes: DARPA ARPANET.

As primeiras redes eram apenas de uso milita e cientifico. Com a criação do Protocolo TCP/IP as redes tornaram-se mais robostas Em 1989, Tim Berners-Lee Cria a World Wide Web. Unindo a tecnologia do TCP/IP com a Linguagem de marcação de texto (HTML)

O que são Clients

Voltar

Cliente no liguajar Web

Cliente é o dispositivo que requisita, recebe e consome informações da internet. Exemplo computador(Cliente) requisita atraves da conexão com a internet informações que estão em outro computador(Servidor) que recebe a requisição e envia a informação que foi pedidam, assim o cliente é o computador que requisita e recebe a informação.

Mas o que tipo de arquivo é tranferido nessa comunicação? Necessariamente são arquivos html, css, javascript, imagens tudo que compoe uma pagina web.

Os navegadores possuem um recurso chamado de cache, que guarda arquivos dos sites visitados. Para que se as paginas requisitas não tiverem mudança ele não requita os arquivo novamente, assim poupando recursos.

Navegadores

São programas que interpretam os arquivos que são recebidos do servidor web (hmtl, css, javascript etc). Os navegadores são desenvolvidos seguindo os padrões W3C, grupo responsavel por padronizar as metricas dos navegadores de internet. Os navegadores também são conhecidos como browsers.

Aplicações Web

Aplicações web são programas que podem ser acessados on-line não sendo nessaria a sua instalação localmente.

Dispositivos Móveis

Atualmente há uma variedade de dispositivos Móveis, exemplo: smartphones, tablets, notebooks e etc. Os dispositivos moveis são reponsaveis pelo maior trafego de dados na internet.

O que são Servers?

Voltar

Sobre Servidores

Os servidores são computadores responsaveis por responder as requisições dos computadores clientes, enviado de si os arquivos que foram requisitados.

São computadores de configuração mais robustas, pois servi a varios computadores.

Há vários tipos de servidores, como por exemplo, servidor de arquivos, servidor de e-mail, servidor Web e etc.

Hardwares e Sofwares em Servidores

Como todos os computadores os servidores possuiem Hardwares (Parte fisica) e sofwares (Parte Lógica). Os servidores possuem Sistema Operacional desenvolvido especificamente para eles. Exemplos de SO para Servidores: Windows Server, Ubuntu, Debian, Fedora e Oracle.

Para atender as requisições feitas ao Servidor, há os Servidores Web, exemplo: Apache e Nginx.

Quando um computador client envia uma requição para um Servidor, essa requesição é recebida pelo Sistema Operacional do Servidor e é repassada para software de Servidor Web para ser processada.

Tipos de Servidores

  • Arquivos - Serve arquivos para rede.
  • Segurança(Firewall) - Um Sofware que é uma barreira de configurações de segunça para evitar acesso exterior a rede.
  • E-mail - armazena e gerencia envio e recebimento de e-mails Web - Serve paginas Web.
  • Database Serve - Contém um banco dados.
  • Proxy - Defina Regras de acesso quanto sites.
  • Servidor Web sem internet - Servi paginas web apenas dentro da rede (intranet).
  • DNS (Domain Name Service) - Faz a conversão do número de endereço ip onde estão hospedos sites para forma de endereço composto por letras facilitando a leitura dos endereços, exemplo: web.dio.me.

Hospedagem de Sites

Colocar os arquivos do site em um servidor, pode ser configurado ou contratado. Ao colocar (subir/upload) dos arquivos para o servidor e devidamente configurado o endereço ficara disponivel para acesso.

HTML não é Linguagem de Programação

HTML(Hyper text Markup Language) é uma linguagem de marcação de texto. É um arquivo de marcação. O texto é marcado com delimitadores e cada delimitador tem uma função especifica. Os navegadores interpreta os delimitados e exibe o texto com a formatação indicada nos delimitadores.

Estrutura Básica do HTML

Voltar

O código HTML é composto por textos e delimitadores chamados tags. Em um arquivo HTML as tags html delimitão todo o código, ou seja, todas as outra tags do html devem estar contidas entre os delimitados (delimitador de abertura da tag) e (delimitador de fechamento da tag).

A estrutura de uma página divide-se em head e body. No head ficam informações que são pré-carregadas antes da exibição da página. No body fica o conteúdo que será exibido ao usuário.

<html>
<head>
<title> Minha Página </title>
<head>
<body>
Meu HTML
<body>
<html>

Falando Sobre Tags

Voltar

Há uma enorme váriedade de tags no HTML cada uma com sua função especifica. Todo o contéudo que fica dentro dos delimitadores de abertura e fechamento da tag se comporta conforme a função da tag.

Exemplos de tags: <i>Este texto fica itálico ao ser mostrado em um navegador</i>

Há algumas tags que não possuem fechamento </> essas tags possuem barra no final. Exemplo de tag sem fechamento:
<img/> <input/>

Atributos das Tags

Voltar

Atributos são propriedades que as tags possuem que podem ser alteradas para se ajustar e ou modificar o comportamento do elemento HTML. Há atributos que são genericos outros são especificos de uma tag.

Exemplo de tag com atributos: <strong style="color:blue"> Este texto fica azul no navegador </strong>

Textos

Voltar

Há várias tags que tem como função formatar o texto. a tag <h1> </h1> por exemplo ao delimiatar um texto destaca que esse texto deve ser tratado com um titulo. As tags de titulos são: h1, h2, h3, h4, h5 e h6. Quando maior o número menor será mostrado o Título.

A tag <p></p> delimita um paragrafo.

A tag <blockquote></blockquote> delimita uma citação longa.

A tag <u></u> é usada para que o texto fique sublinhado.

A tag <i></i> torna o texto itálico.

Para Marcar um texto usar-se a tag <mark></mark>.

Para elevar um caractere usar-se a tag <sup></sup>.

Lista Ordenadas e Não Ordenadas

Voltar

Para criar uma lista ordenada usa-se a tag <ol></ol>.

Para criar uma lista não ordenada usa-se a tag <ul></ol>.

As tags de criação de listas são usadas em cojunto com as tags de listagem dos item da lista <li></li> tendo uma relação hierarcca as tags li ficam dentro da delimitação das tags ol, sendo chamadas de tags filhas, sendo assim a tag ol ou ul sendo consideradas as tags pais.

Exemplo:
<ol>
<li>item 1</li>
<li>item 2</li>
</ol>
<ul>
<li>item 1</li>
<li>item 2</li>
</ul>

Links

Voltar

Para criar hiperlinks ou links usamos a tag , quando se liga uma pagina a outra diz-se que uma página está ancorada a outra, daí a tag de link ser representada por uma letra "a". O link pode levar para outro lugar na mesma página.

Exemplo de aplicação da tag :

<a> href="http//dio.me" title="Clique aqui e confira" target="_blank" >Dio.me<a>

O atributo href define o endereço para onde o link aponta.

O atributo target pode ser usado para definir se a pagina destino será aberta em uma nova aba.

O atributo title define um balão que abre com uma mensgem ao se passar o ponteiro do mouse por cima do link.
